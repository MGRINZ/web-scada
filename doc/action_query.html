action=query

Query data from database

Parameters

list string

List type of which data should be queried.

Acceptable values: vars

Lists

vars

Parameters

variable string

Optional: Query only variables of desired type.

Acceptable values: I, Q, R, AI
order string

Optional: Order variables by their addresses or labels..

Acceptable values: var, label

Default: var

Return

var string

Type of variable
address int

Address of variable
type string

Data type of variable
label string

Label of variable

 
[
  {
    "var": "string",
    "address": "int",
    "type": "string",
    "label": "string"
  }
]

Examples

List all variables

/api.php?action=query&list=vars

[
  {
    "var": "AI",
	"address": 1,
	"type": "INT",
	"label": "labelAI1"
  },
  {
    "var": "I",
	"address" :3,
	"type": "BOOL",
	"label": "labelI3"
  },
  {
    "var": "R",
	"address": 1,
	"type": "DINT",
	"label": "labelR1"
  }
]

action=read

Reads variable value from database

Parameters

label string

Read variable value by its label. Cannot be used with var and address.

Acceptable values: any string
var string

Read variable value by its address. Must be used with address. Cannot be used with label.

Acceptable values: I, Q, R, AI
address int

Read variable value by its address. Must be used with var. Cannot be used with label.

Acceptable values: integer value in range [0, 65535]

Return

value double

Value of variable

 
{
  "value": "double"
}

Examples

Read variable called "labeled_var"

/api.php?action=read&label=labeled_var

{
  "value": 10
}

Read variable %R1

/api.php?action=read&var=R&address=1

{
  "value": 10
}

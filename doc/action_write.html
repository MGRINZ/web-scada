action=write

Writes new variable value to database

Parameters

label string

Write variable value by its label. Cannot be used with var and address.

Acceptable values: any string
var string

Write variable value by its address. Must be used with address. Cannot be used with label.

Acceptable values: Q, R
address int

Write variable value by its address. Must be used with var. Cannot be used with label.

Acceptable values: integer value in range [0, 65535]
type string

Data type of variable. Has no effect when used with label.

Acceptable values:

  • var=Q: BOOL
  • var=R: INT, DINT, LINT, UINT, UDINT, ULINT, REAL, LREAL
value double

Value of variable

Acceptable values: any double value

Return

result string

Result of writting.

ok – write successfull

error – write error

 
{
  "result": "string"
}

Examples

Write 10 to variable called "labeled_var"

/api.php?action=write&label=labeled_var&value=10

{
  "result": "ok"
}

Write 123 as INT to variable %R1

/api.php?action=write&var=R&address=1&type=INT&value=123

{
  "result": "ok"
}
